## 🛠️ 서비스 주요 기능
- **로그인/회원가입**
    - 약관 동의 및 프로필 설정 가능
    - 설문을 통해 여행자 유형 분석 제공

- **여행 일정 관리**
    - 여행의 제목 및 기간 선택 후 일정 생성
    - 동행자 초대 및 공동 일정 계획 가능
    - 소유자만이 여행 삭제 가능하며, 동행자 목록에서도 자동 삭제

- **실시간 장소 관리**
    - 장소 선택 후 이모지, 날짜, 시간, 링크, 메모 추가 가능
    - 장소의 순서 변경 및 삭제 가능
    - 실시간 업데이트를 통해 동행자와 공유

- **친구 관리**
    - 회원가입 시 5자리 코드 자동 생성, 이를 이용한 친구 추가 가능
    - 알림을 통해 친구 추가 요청 확인 가능

## ⁉️ 프로젝트를 진행하면서 생긴 문제점 및 해결방안
* **데이터베이스 활용**
  * 일정 관리 앱이라는 특성상 데이터를 잘 활용해야 하는데, 데이터 활용에 어려움을 겪음.
  * 해결 - 시간을 들여서 firebase공식 문서나 다른 사람들이 작성한 글을 살펴보며 해결. 

* **여행 작성 페이지 동작 매커니즘**
  * 구상했던 디자인과는 다르게 지도와 검색 부분에 의해 디자인에 맞게 개발이 힘든 영역들이 있어서 어려움을 느낌.
  * 해결 - 디자인을 수정.
    
* **구글맵 검색 오류**
  * 검색 후 일정을 적는 도중에 계속해서 에러 발생.
  * 해결 - GooglePlacesAutocomplete에서 debounce를 0 이상으로 설정.
  
* **Tensorflow 모델 로드 오류**
  * tensorflow와 expo의 충돌로 인한 문제.
  * 해결 - FastAPI의 별도의 서버를 구축하고, 여행분류 모델을 API로 서빙.
